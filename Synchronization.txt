Pattern:
-------------------------------------------------
**Add Step:**  Add step to Steps; Add step to clonedSteps
**Remove Step:** Delete step from Steps; delete step from clonedSteps
**Edit Step:** **Remove Step + Add Step** at same index *or* update step in Step and clonedSteps


**Suggest Step:** Add step to clonedSteps setting suggestedStep: True
**Suggest Remove:** set suggestedStep: True on existing Step (and add strike-through); set suggestedStep on clonedStep (and add strikethrough)
**Suggest Edit:** **Suggest Remove + Suggest Step**


**Accept Step:** Add suggestedStep to Steps setting suggestedStep: False; clonedSteps set suggestedStep: False 
**Reject Step:** Delete step from clonedSteps (remove locally)
